# 📚 API ENDPOINTY
================================

## Admin API (vyžaduje autentizaci)
POST   /api/admin/auth/v2/login      - Přihlášení
POST   /api/admin/auth/v2/refresh    - Obnovení tokenu
GET    /api/admin/articles           - Seznam článků
POST   /api/admin/articles           - Vytvoření článku
PUT    /api/admin/articles/[id]      - Úprava článku
DELETE /api/admin/articles/[id]      - Smazání článku

GET    /api/admin/categories         - Seznam kategorií
POST   /api/admin/categories         - Vytvoření kategorie
PUT    /api/admin/categories/[id]    - Úprava kategorie
DELETE /api/admin/categories/[id]    - Smazání kategorie

POST   /api/admin/media/upload       - Upload média
GET    /api/admin/media              - Seznam médií
DELETE /api/admin/media/[id]         - Smazání média

GET    /api/admin/analytics          - Statistiky
GET    /api/admin/settings           - Nastavení systému
PUT    /api/admin/settings           - Úprava nastavení

## Veřejné API (bez autentizace)
GET    /api/admin/public/articles    - Publikované články
GET    /api/admin/public/categories  - Aktivní kategorie
